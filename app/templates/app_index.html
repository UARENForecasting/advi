ADVI is an interactive plot showing output from the Weather Research and Forecasting model run by the University of Arizona UA Renewable Power Forecasting Group and the UA Department of Hydrology and Atmospheric Sciences . Click on a model variable above to get started.

The site was produced using the Bokeh visualization library with map tiles provided by Stamen Design. The open-source code for this site can be found on GitHub. Please report any issues or bugs on the issue tracker .
