ADVI is an interactive plot showing output from the Weather Research and Forecasting model run by the University of Arizona UA Renewable Power Forecasting Group and the UA Department of Hydrology and Atmospheric Sciences. The site was produced using the Bokeh visualization library with map tiles provided by Stamen Design.

Use the left and right arrows to change the forecast hour. Clicking on a the map will display the current value of the selected location in the selected value text box. The histogram and mean value are based only on the data currently displayed on the map.
